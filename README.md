# Swipe Media Cleaner

Приложение для удобной очистки галереи на телефоне, вдохновленное ClipCleaner.

## Функционал

### ✅ Реализовано (v0.1)

- **Запрос разрешений** - автоматический запрос доступа к фотографиям
- **Группировка по месяцам** - все фото сгруппированы по месяцам и годам
- **Карточки месяцев** - красивые карточки с информацией:
  - Название месяца на русском
  - Год
  - Количество фотографий
  - Общий размер
- **Сортировка** - новые месяцы отображаются сверху
- **Pull-to-refresh** - обновление списка свайпом вниз

### 🚧 В разработке

- Экран просмотра фото конкретного месяца
- Функционал свайпа для удаления фото
- Корзина для временного хранения удаленных фото
- Статистика и аналитика использования памяти

## Технологии

- **Flutter** 3.10.6
- **Riverpod** 2.4.x - state management
- **photo_manager** - доступ к галерее
- **permission_handler** - управление разрешениями

## Разрешения

### Android
Приложение запрашивает следующие разрешения:
- `READ_EXTERNAL_STORAGE` (для Android <= 12)
- `READ_MEDIA_IMAGES` (для Android >= 13)
- `WRITE_EXTERNAL_STORAGE` (для удаления фото)

### iOS
- `NSPhotoLibraryUsageDescription` - доступ к фото
- `NSPhotoLibraryAddUsageDescription` - управление фото

## Использование

1. При первом запуске предоставьте доступ к фотографиям
2. Просмотрите список месяцев с фотографиями
3. Выберите месяц для просмотра фото (в разработке)
4. Используйте свайп для удаления ненужных фото (в разработке)
